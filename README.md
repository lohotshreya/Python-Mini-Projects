# 🐍 Python Mini Projects

A collection of beginner-friendly Python mini projects built to strengthen core programming concepts through hands-on practice. These projects focus on logic building, user interaction, and clean code structure — perfect for beginners and early-stage learners.

---

## 📁 Projects Included

### 1️⃣ Chatbot
A simple rule-based chatbot that responds to user inputs.
* **Concepts used:** Input/output, Conditional statements, String manipulation

### 2️⃣ Rock Paper Scissors
A classic Rock–Paper–Scissors game played against the computer.
* **Concepts used:** `random` module, Conditional logic, Loops

### 3️⃣ Calculator
A basic calculator supporting arithmetic operations.
* **Concepts used:** Functions, User input parsing, If-else conditions

### 4️⃣ Number Guessing Game
The computer randomly selects a number, and the user guesses until correct.
* **Concepts used:** `while` loops, Random number generation, Conditional statements

### 5️⃣ To-Do List (CLI)
A command-line-based to-do list application to manage daily tasks.
* **Features:** Add tasks, View tasks, Delete tasks
* **Concepts used:** Lists, Loops, Menu-driven programming

### 6️⃣ Password Generator / Strength Checker
Generates secure random passwords or checks whether a given password is weak, medium, or strong.
* **Concepts used:** `string` operations, Length checking, Conditional logic

### 7️⃣ Contact Book
An application to store, view, search, and delete contact details.
* **Concepts used:** Dictionaries, Functions, User interaction

### 8️⃣ Quiz Game
A simple multiple-choice quiz game with automatic scoring at the end.
* **Concepts used:** Dictionaries/Lists of tuples, Loops, Score tracking

### 9️⃣ Currency Converter
Converts amounts between different global currencies using fixed/hardcoded conversion rates.
* **Concepts used:** Data structures (dictionaries), Floating-point arithmetic, User input validation

### 🔟 Desktop Notifier
Generates custom system desktop notifications at set intervals or events.
* **Concepts used:** External libraries (`plyer`), Time modules, Background loops

### 1️⃣1️⃣ Dice Roller Simulator
Simulates rolling one or more dice with random outcomes and options to roll again.
* **Concepts used:** `random` module, `while` loops, Functions

### 1️⃣2️⃣ QR Code Generator
Generates and saves custom QR codes from user-provided URLs or text strings.
* **Concepts used:** External libraries (`qrcode`, `Pillow`), File saving

### 1️⃣3️⃣ Digital Metronome
A functional metronome that plays a precise sound or visual pulse at a user-defined BPM (Beats Per Minute).
* **Concepts used:** `time` module, Audio/Beep playback handling, Input math conversions

### 1️⃣4️⃣ Weather App
Fetches and displays real-time weather information for any user-inputted city using an external API.
* **Concepts used:** API integration, `requests` library, JSON parsing, Error handling

### 1️⃣5️⃣ Expense Tracker
A command-line tool to log expenses, track spending categories, and persist data between runs.
* **Concepts used:** File handling (`json` / `csv`), Data serialization, Data aggregation

### 1️⃣6️⃣ Unit Converter (GUI)
A graphical desktop application to convert units like temperature, weight, and distance.
* **Concepts used:** GUI design (`tkinter`), Event-driven programming, Type conversion

### 1️⃣7️⃣ Text-Based Adventure Game
A choice-based interactive storytelling game where user decisions fork the game's outcome.
* **Concepts used:** Nested conditional structures, String sanitization, Structural control flow

### 1️⃣8️⃣ Web Scraper (Simple)
Extracts headlines, text quotes, or links from public live web elements and formats them.
* **Concepts used:** HTTP requests, HTML parsing (`beautifulsoup4`), DOM inspection

### 1️⃣9️⃣ Markdown to HTML Converter
Reads a local Markdown file, converts basic syntax features (`#` and `**`) into HTML structural tags, and outputs an openable webpage.
* **Concepts used:** Regular expressions (`re`), Advanced file streams (`open()`), Sub-string replacing

### 2️⃣0️⃣ Automated Email Sender
A network script that automates sending plain-text alert emails securely to a designated client inbox.
* **Concepts used:** Network protocols (`smtplib`, `ssl`), Security authorization protocols, Environment variables

### 2️⃣1️⃣ Frequency Counter & Text Analyzer
Analyzes large paragraphs or text files to count word distributions, find structural density, and filter out noise words.
* **Concepts used:** `collections.Counter`, Data normalization techniques, List comprehensions

### 2️⃣2️⃣ Caesar Cipher Cryptography Tool
An algorithmic implementation to encrypt and decrypt basic text messages by shifting characters down the alphabet.
* **Concepts used:** Modular arithmetic, ASCII byte conversions (`ord()`, `chr()`), String transformations

### 2️⃣3️⃣ International Space Station (ISS) Tracker
Pulls live location coordination frames tracking the exact geospatial positioning of the ISS orbiting overhead.
* **Concepts used:** API requests, Loop-driven background polling, Data structural parsing

### 2️⃣4️⃣ Fake Profile Generator
Connects directly to safe open mockup endpoints to dynamically construct realistic, unauthenticated client database profiles for software testing.
* **Concepts used:** Network requests, Clean terminal format compilation, Handling anonymous JSON arrays

### 2️⃣5️⃣ Markdown Link Extractor
Scans through documents to cleanly extract and surface hidden hyperlink arrays embedded inside structural markdown boundaries.
* **Concepts used:** Advanced string patterns, String pattern filtering, `re.findall()` logic

### 2️⃣6️⃣ Typing Speed Test
Calculates keyboard typing execution metrics, showing individual tracking accuracy alongside Words-Per-Minute (WPM) scoring calculations.
* **Concepts used:** `time` subtraction models, Precise execution intervals, Algorithmic evaluations

### 2️⃣7️⃣ Word Unscrambler (Anagram Solver)
Deciphers jumbled scrambles of text arrays and instantly cross-checks possible combinations against matching definitions.
* **Concepts used:** `itertools.permutations`, Set comparison logic, Modular combinatorial mathematics

### 2️⃣8️⃣ Countdown Timer & Alarm
A command-line terminal clock tool managing uniform, cascading second countdown intervals with real-time screen display clearing.
* **Concepts used:** Python line rewriting flags (`\r`), Dynamic mathematical breakdowns (`divmod()`), Blocking loops

### 2️⃣9️⃣ Image Metadata Viewer (Exif Reader)
Reads an image asset container from a folder and maps internal attributes like encoding format, resolution scale, and color spaces.
* **Concepts used:** Python Image Library (`Pillow`), Asset stream unpacking, Binary attribute scanning

### 3️⃣0️⃣ Text File Compressor
Emulates lossless run-length file data optimization structures by mapping duplicate arrays into mathematical tokens.
* **Concepts used:** Run-length string evaluations, Array caching indices, Value tracking

### 3️⃣1️⃣ Random Joke Displayer
Pulls real-time programming jokes directly from a secure cloud platform via an open API framework.
* **Concepts used:** Data parsing loops, Endpoint parameters filtering, Error-resilient parsing

### 3️⃣2️⃣ Directory Tree Visualizer
Constructs a beautiful, structured text layout diagram mapping out complex directory system loops.
* **Concepts used:** System file navigation modules (`os`), Recursive call trees, String branching arrays

* **Language:** Python 3
* **Built-in Standard Libraries:** `random`, `time`, `json`, `csv`, `tkinter`, `os`, `re`, `smtplib`, `ssl`, `collections`, `itertools`
* **External Dependencies:** `requests`, `qrcode`, `plyer`, `beautifulsoup4`, `pillow`

To install all required external packages across the entire collection at once, run:
